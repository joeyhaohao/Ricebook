# RiceBook

RiceBook is a Rice social networking site to post ideas, follow users and chat online.

## Tech Stack
* Angular, Ajax
* Nodejs, Express, MongoDB
* Redis
* OAuth
* WebSocket

## Main Features
* Facebook/Rice login
* Post ideas and images
* User Profile management
* Follow users
* Online chat
